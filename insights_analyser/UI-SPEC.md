# UI-SPEC: Insights Analyser Dashboard

## Vision
A high-fidelity, spatial dashboard for product teams to digest weekly app store sentiment. The interface uses "Antigravity" design principles: weightless transitions, glassmorphism, and a focused teal/ink palette.

## Core Components

### 1. The Pulse Header
- **Title**: "Weekly Product Pulse"
- **Context**: Date range (e.g., "Apr 15 - Apr 22") and App Name ("SBI Mutual Fund").
- **Action**: "Refresh Analysis" button (Teal #00B5B7).

### 2. Theme Grid (The Wedge)
- **Layout**: 3-column grid of glassmorphism cards.
- **Interaction**: On hover, cards lift slightly. On click, they expand to show real user quotes.
- **Card Content**:
    - Theme Title (e.g., "KYC Delays")
    - Sentiment Indicator (Icon/Color)
    - Summary (2 sentences)

### 3. Action Intelligence Panel
- **Content**: The 3 "Action Ideas" generated by the LLM.
- **Visual**: Numbered list with checkmark icons.

### 4. Communication Loop
- **Component**: "Email Team" floating action button.
- **Interaction**: Opens a side-panel with the pre-written email draft and a "Copy to Clipboard" feature.

## Design Tokens (Refined)
- **Glassmorphism**: `background: rgba(255, 255, 255, 0.7); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2);`
- **Shadows**: Deep, soft shadows for the "lifting" effect.
- **Transitions**: 0.4s cubic-bezier easing for all interactions.

## Technical Implementation
- **Framework**: React 19 + Vite.
- **Animation**: GSAP for the "Spatial" entrance of cards.
- **Data Source**: Fetches from `insights_analyser/output/weekly_note.md` (parsed) or a structured JSON.
